title: Pay
description: Pay someone some credits or redeems
---

Use the /pay command to transfer credits or redeems to another user.
You must reply to the message of the user you want to pay.

Usage examples:
/pay 200 Credits , 
Pay 200 credits to the user you replied to.

/pay 2 Redeems , 
Pay 2 redeems to the user you replied to.

Specify the amount and the currency type (either credits or redeems).
