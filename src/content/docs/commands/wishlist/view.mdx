---
title: Wishlist view
description: View your wishlist
---

Use the command:
/wishlist view [page]

Run /wishlist view to see the first page of your Pokémon wishlist.
To view a specific page, add the page number after the command, for example /wishlist view 3 to see the 3rd page.
This command helps you easily track and view all the Pokémon you have added to your wishlist, enabling better management and targeting of your desired Pokémon.
