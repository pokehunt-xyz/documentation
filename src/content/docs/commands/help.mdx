---
title: Help
description: Get help using PokéHunt
---

Use the command:
/help [command]
To view a list of all available commands, simply use /help.

To get detailed help for a specific command, add the command name after /help, for example:
/help nickname , 
This will show how to use the /nickname command.

This helps you quickly learn how to use PokéHunt’s commands and get assistance on specific features.
