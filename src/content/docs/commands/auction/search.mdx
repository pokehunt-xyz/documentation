---
title: Auction search
description: Search for Pokémon listed on the auction
---

Use the command to search Pokémon available on the auction:
/auction search [page=1] , 
You can specify the page number to browse through the auction.

To search for a specific Pokémon by name, use:
/auction search bulbasaur , 
This will show all Bulbasaurs currently on auction.

To view the latest page of auction , use:
/auction search -1 , 
This command helps you quickly find and browse Pokémon available for bidding in the auction.
