# Buying

<Steps>

1. First find a Pokémon you want to buy using [/market search](/commands/market/search)

2. Then use [/market view](/commands/market/view) to see details about this Pokémon

3. If you wish to buy it, use [/market buy](/commands/market/buy)

</Steps>

# Selling

To sell your Pokémon use [/market create](/commands/market/create) and set a fixed price.

To view all your Pokémon listed for sale, use [/market listings](/commands/market/listings).
