---
title: Dex
description: Get information about a specific Pokémon
---

Use the command:
/dex <pokemon name> , 
to get detailed information about a specific Pokémon.

Example:
/dex bulbasaur
This command provides comprehensive data including the Pokémon’s base stats, type, weaknesses, evolution details, and other relevant information. 
It helps you learn about any Pokémon in the game, including how to use them effectively .
